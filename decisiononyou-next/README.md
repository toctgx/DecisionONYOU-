# DecisionONYOU Next.js Workspace

이 디렉터리는 기존 정적 프로토타입을 Next.js + Supabase 구조로 이관하기 위한 새 앱 작업 공간입니다.

## 현재 상태
- Next.js App Router + TypeScript 기반 골격 생성 완료
- 기본 랜딩 화면을 DecisionONYOU 마이그레이션 안내 화면으로 교체 완료
- 이후 단계에서 홈 피드 / 질문 등록 / 내 페이지 / Supabase 연동을 순서대로 이관 예정

## 실행
```bash
npm install
npm run dev
```
